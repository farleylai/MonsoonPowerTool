Voice Activity Detection

15 December 2015

Three Voice Activity Features

Short-Term Energy (E)

Sum of squared float sample values.

\(Energy_{primthreshold} = 40\)

Dominant Spectral Frequency Component (F)

The frequency of the maximum spectrum magnitude.

\(F_{primthreshold} = 185 (Hz)\)

Spectral Flatness Measure (SFM)

\(SFM_{db} = 10log_{10}(G_m/A_m)\)

\(SFM_{primthreshold} = 5\)

\(G_m = \big(\prod\limits_{n=0}{n-1} a_i \big){1/n} = \exp \big[ \frac{1}{n} \sum\limits_{n=0}^{n-1}\ln a_i \big] \)

pxx = periodogram(a);
%figure(2)
%plot(pxx)
num=geomean(pxx);
den=mean(pxx);
spf=num/den ;

Power Spectral Density Estimate using FFT (PSD)

N = length(x);
xdft = fft(x);
xdft = xdft(1:N/2+1);
psdx = (1/(Fs*N)) * abs(xdft).^2;
psdx(2:end-1) = 2*psdx(2:end-1);
freq = 0:Fs/length(x):Fs/2;

plot(freq,10*log10(psdx))
grid on
title('Periodogram Using FFT')
xlabel('Frequency (Hz)')
ylabel('Power/Frequency (dB/Hz)')
