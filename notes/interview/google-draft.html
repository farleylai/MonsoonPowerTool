Q. Infinite stream of numbers. How do you predict what will be the next number?

Collect a sufficiently long sequence of numbers and compute the probabilies for each number.

Q. How to remove duplicate line in a very large text file?

Compute and put the hashes for each line in a set. If the hash is already in the set, the corresponding line is removed. If the text file is large, we can process in a divide and conquer fashion. Recursively divide in halves and the leaf nodes can be process in parallel. The merge is bottom up and also in parallel. The overall time complexity would be O(nlognlogn).

Q. Find if a given binary tree has duplicate sub trees.

Just two leaf nodes of a parent do not count as subtree in this question. Recursively compute the hashes of both children and check if the same.

Q. Find all possible combinations for a string of 0, 1 and ?.

The question mark can be either 0 or 1. Recursively print the string with the question mark set to 0 and 1.
