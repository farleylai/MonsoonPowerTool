Multivariate Linear Regression

23 February 2016

Feature Scaling

\(-1 \le x_i \le 1\)

Mean Normalization

Replace \(x_i\) with \(x_i - \mu_i\) and then apply feature scaling by dividing by (max - min)

Convergence

If the cost function decreases less than \(10^{-3}\) in one iteration

Learning Rate

Sufficiently large and make sure the cost function is decreasing by choosing 0.001, 0.003, 0.01, 0.03, 0.1…

Polynomial Regression

Set high order features as new variables to approximate with linear regression

Normal Equations

\( \theta = (XT X){-1} X^T y \) where \(X\) is the design matrix and no need for feature scaling or mean normalization

Gradient Descent (GD) vs. Normal Equation (NE)

GD works faster when \(n\) is as large as \(104\) while NE is slow in \(O(n3)\)

Non-Invertible

Feature multiplication \(X_TX\) might be invertible.

  • Redundant features that are linear dependent

  • Too many features

    • Use regularization
